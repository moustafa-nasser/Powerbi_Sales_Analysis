## üîç Business Problem
Companies often struggle to monitor Key Performance Indicators (KPIs) in real-time and understand customer behavior effectively. This dashboard aims to solve these challenges by answering critical business questions:
1. What are the total sales and profits across different time periods?
2. Which products contribute the most to revenue (Top Selling Products)?
3. How does sales performance vary across different regions?
4. What are the Year-over-Year (YoY) and Month-over-Month (MoM) growth trends?

---

## ‚öôÔ∏è Process & Methodology
The project was executed through the following stages:

1.  **Data Extraction & Transformation (ETL):**
    * Imported raw data from various sources (CSV/Excel).
    * Cleaned data and handled missing values using Power Query editors.
    * Standardized date formats and data types for consistency.

2.  **Data Modeling:**
    * Designed a robust Data Model connecting Fact tables to Dimension tables.
    * Established proper relationships (One-to-Many) to ensure accurate filtering.

3.  **DAX Calculations:**
    * Developed custom measures for dynamic analysis, such as: `Total Sales`, `Total Profit`, `Profit Margin %`, and `YTD Sales`.

4.  **Dashboard Design:**
    * Designed a user-friendly interface with a focus on UX/UI principles.
    * Utilized appropriate visualizations (Bar Charts, Line Charts, Maps, Slicers).

---

## üìä Key Features
* **Interactive Slicers:** Filter data dynamically by Year, Month, Product Category, or Region.
* **KPI Cards:** Instant view of vital metrics (Total Revenue, Net Profit, Order Count).
* **Trend Analysis:** Visualizations displaying sales performance over time to identify seasonality.
* **Geographical Analysis:** Interactive maps highlighting sales distribution globally.
